NOTICE

The use of these scripts depends on the creation of the /etc/netns/vpn and /etc/netns/tor directories.

In the case of the former, both /etc/resolv.conf and /etc/resolvconf.conf need to be copied into the /etc/netns/vpn directory, to avoid disrupting DNS on the host.

In the case of the latter, I don't know, I haven't done the TOR part yet.
